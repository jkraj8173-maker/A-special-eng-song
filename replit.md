# Love Lyrics Site - For Aradhya

## Overview
A romantic love lyrics website made by Jeet for Aradhya. Features animated text, background music, floating fireflies, butterfly burst effects, and beautiful glowing text animations.

## Project Structure
- `src/app/` - Next.js app router pages and layouts
- `src/components/` - React components (Background, FirstScreen, LyricsScreen, etc.)
- `src/components/ui/` - UI utilities (text-animate)
- `src/lib/` - Utility functions
- `public/` - Static assets (audio, gifs)

## Features
- Personalized message for Aradhya with glowing gradient name
- Floating fireflies (purple and yellow Jugnu)
- Butterfly burst animation on button clicks
- Sparkle effects throughout
- Song lyrics with purple glow animations
- "I love u bubu" ending message

## Tech Stack
- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion (animations)
- Lucide React (icons)

## Development
- Run: `npm run dev -- -p 5000 -H 0.0.0.0`
- Build: `npm run build`
- Start production: `npm run start`

## GitHub Hosting
To deploy on GitHub Pages:
1. Run `npm run build`
2. The output is in the `.next` folder
3. For static export, you may need to add `output: 'export'` in next.config.mjs
